## 🍟 Color Filpper
> 랜덤으로 배경색을 바꿔주는 프로젝트
- simple button을 누르면 배열에 지정되어 담겨 있는 컬러들이 랜덤으로 바뀌며 배경색을 변경
- hex button을 누르면 배열에 담긴 16진수가 랜덤으로 hex color code를 만들어 배경색이 변경 

## 🍟 이 프로젝트에서 다루는 주요 개념

- array
- document.getElementById()
- document.querySelector()
- addEventListener()
- document.body.style.backgroundColor
- Math.floor()
- Math.random()
- array.length
